Diagnostic logging

What

The Vocalizer engine publishes a lot of information useful to assist in debugging or to collect performance metrics. A diagnostic logger captures this log information and may write it to a text file. It is possible to configure this diagnostic logger through a header file.

How

Setup is simple.

  • Create the directory ve under ~\common\speech.
  • Copy the header file ve_logging.hdr from ~\sample\logging to ~\common\speech\ve.
  • Edit ve_logging.hdr to set the log level to your needs.

The different diagnostic levels are

  • 0: log errors
  • 1: also log warnings
  • 2: also log computational statistics (memory and processing footprints)
  • 3: also log debugging info
  • 4: also log trace info
